# vue-news

- Hack News API를 이용한 list 페이지 구성
- router, axios, mixin, async, await 등 사용
- spinner 컴포넌트 구현
- Chart.js를 활용한 컴포넌트 구현
- netlify를 이용한 간이 배포 실시
